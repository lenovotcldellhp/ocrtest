# captcha_recognize
哈工大（威海）模式识别

采用Tesseract库，实现Python接口和Python图像预处理，以进行验证码的识别。

验证码测试样例来自百川PT。

详细PPT见captcha_recognize.pdf

安装说明：

* 需安装好Tesseract，并配置好环境变量

* 需要Python3运行环境

* 安装好后即可运行
